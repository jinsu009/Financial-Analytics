# 💰 Financial Analytics (수진의 돈관리)

개인 자산 현황과 예산 관리를 한눈에 파악할 수 있는 **React 기반 가계부 대시보드**입니다. 
Vite와 shadcn/ui를 활용하여 현대적이고 직관적인 UI를 구현했습니다.

---

## ✨ 주요 기능

- **실시간 예산 추적**: 남은 예산 수치에 따라 프로그레스 바의 색상이 변하며 위험 상태를 직관적으로 표시합니다.
- **반응형 사이드바**: `shadcn/ui`의 Sidebar 시스템을 활용하여 접기/펴기 및 툴팁 기능을 지원합니다.
- **데이터 시각화**: `Recharts`를 이용한 소비 카테고리별 비중 차트를 제공합니다.
- **반응형 레이아웃**: 데스크톱과 태블릿 등 다양한 화면 크기에 최적화된 Grid 레이아웃을 적용했습니다.

---

## 🛠️ 기술 스택

### Frontend
- **Framework**: React 18 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, lucide-react (Icons)
- **Components**: shadcn/ui (Radix UI 기반)
- **Charts**: Recharts
